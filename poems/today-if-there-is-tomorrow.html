Today If There Is Tomorrow
scroll
Today if there is tomorrow If there was a day it was tomorrow and if it was tomorrow, it was a day
Where did the birds go when the rains fall without end? Where do we go when we find our heart? Where do we go when we wonder about tomorrow?
Where do we go when our hearts quail from fright?
Where do we go when we wonder mystery and forget heart. Where do we go when we leave this life?
Into tomorrow, I say, and fight me if I’m wrong.
